[ About ]

With this library you can play SunVox songs and synths in your applications.
SunVox is a small, fast and powerful modular synthesizer with pattern-based sequencer (tracker):
http://www.warmplace.ru/soft/sunvox

[ Structure ]

headers - SunVox C/C++/Pixilang header files.
ios - SunVox library for iOS + sample projects.
linux - SunVox library for Linux + sample projects.
linux_android - SunVox library for Android + sample projects.
osx - SunVox library for OSX + sample projects.
pixilang - example of using SunVox library in Pixilang application.
resources - SunVox songs and modules for the sample projects.
windows - SunVox library for Windows + sample projects.
